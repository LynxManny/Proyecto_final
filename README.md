Es un juego de ahorcado con interfaz gráfica que permite, agregar palabras a una lista, incluye un dibujo que se va completando según los errores cometidos.
Se aplico la interfaz gráfica de Tkinter, incluyendo la creación de la ventana principal, los botones de acción, y la función para dibujar la horca y el muñeco.
Aprendí a utilizar Tkinter para crear interfaces gráficas, el manejo de botones. También en la estructuración del código y la organización de funciones.
